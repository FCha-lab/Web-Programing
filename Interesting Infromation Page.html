관심 기술

A.I.(Artificial Intelligence)

설명 : 인간이 할수 있는 모든 지능적 사고를 컴퓨터가 할수 있게 만드는 기술
기술 근황 : 제한적인 환경의 약인공지능은 많은 발달이 있었다. 하지만 주어진 데이터를 답습하는 것 이상으로 추론하고 사고하는 강인공지능은 여전히 발전히 더딘 상태이다.
적용 가능 분야 : 게임 제작 전분야
  • 에셋(Assets) 제작
  • 기능성 API 강화
  • 가상현실의 고지능 캐릭터, 아바타 현실화

빅데이터(Big Data)

설명 : 일반적인 DB가 관리할 수 있는 양을 초과하는 데이터셋
기술 근황 : DB, 인공지능과 밀접한 연관관계를 유지하는 중이다. AI 기술의 핵심으로 평가받고 있으며 전처리와 연관분석 알고리즘의 효율성을 끌어올리는데 집중하고 있다.
적용 가능 분야 : BM 분석과 인공지능 활용
  • 유저의 행동패턴을 분석하여 최적의 콘텐츠, BM 도출
  • NPC 및 몬스터의 학습자료를 도출

VR(Virtual Reality) & AR(Augmented Reality)

설명 : VR-컴퓨팅 파워를 통해 실제 존재하지 않는 가상의 공간을 만들어내는 기술, AR-컴퓨팅 파워를 통해 실제 존재하는 공간 위에 가상의 그래픽 오브젝트를 만들어내는 기술
기술 근황 : VR은 디스플레이의 발전이 지속적으로 이루어지고 있고 AR도 다양한 분야에서 활용되고 있으나 막상 그 활용 영역은 제한적인 편이다.
적용 가능 분야 : VR, AR 기술을 활용한 게임 제작
  • 가상 현실에서 육체를 활용하여 조작되는 더 생동감 있는 게임
  • 모바일 디스플레이 환경에 최적화된 AR 게임
  • 가상현실을 교두부로 활용되는 플랫폼 제작

관심 기업

펄어비스

설명 : 2010년에 설립된 게임 회사
회사 키워드 : 집요, 야성, 신뢰
개발 게임 : 검은 사막을 비롯한 MMO계열의 게임에 두곽을 보이고 있다.
  • 검은 사막
  • 붉은 사막(발매예정)
  • 도깨비(발매예정)

라인게임즈

설명 : 라인에게 인수된 산하계열사의 게임 개발사
회사 키워드 : 재밌는 세상, 또다른 세상
개발 게임 : 모바일에 강세를 보이고있으며 한국 게임사치곤 드믈게 콘솔에도 개발인력을 쏟고있다.
  • 대항해시대 온라인
  • 데스티니 차일드
  • 베리드 스타즈

네오위즈

설명 : 1997년 인터넷 프로그램 관련 사업으로 창립하여 게임으로 발을 넓힌 게임 개발사
회사 키워드 : 모두를 놀라게 할 게임, 창의적인 아이디어, 비전의 공유
개발 게임 : 웹보드게임에서 수입을 많이 얻고 있으며 최근 콘솔게임 쪽에 큰 투자를 하고있는 상황이다.
  • 피망 섯다
  • DJ MAX
  • P의 거짓(발매예정)
